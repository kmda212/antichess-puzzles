♟ Anti-Chess Puzzle Creator

A browser-based tool to design, record, and review Anti-Chess puzzles — all stored locally in your browser with no server required.

🧩 Puzzle Creator
📋 Solution Recorder
💾 JSON / PGN Export
🎯 Anti-Chess Rules Enforced

⚡ Quick Reference — Three Modes

⚙️
Setup Mode Place / remove pieces freely. Choose whose turn it is.
⏺️
Record Mode Anti-chess rules enforced. Every move is captured as the solution.
▶️
View Mode Step through a saved puzzle's solution move-by-move.
💾
Auto-save Puzzles persist in localStorage between sessions.

📜 Anti-Chess Rules (enforced during recording)

Captures are mandatory — you must capture if any capture is available
When multiple captures exist, you choose which to make
The King has no special status — it can be captured like any piece
No castling in Anti-Chess
En passant is legal
Pawns can promote to any piece including King
Goal: lose all your pieces, or have no legal moves
No pieces / no moves remaining = WIN
Phase 1 — Creating a Puzzle
1
Start a New Puzzle
Resets the board to the standard starting position and enters Setup Mode

When you open the app for the first time, you'll see the chess board in Setup Mode (green badge in the top-right). The board starts in the standard chess position. Click ＋ New Puzzle in the left sidebar to begin.

  • 1Click ＋ New Puzzle in the Workflow panel (top of left sidebar)
  • 2The board resets to the standard starting position
  • 3The mode badge changes to ⚙ Setup Mode
Anti-Chess Puzzle Creator — Setup Mode
💡 Clicking New Puzzle clears any unsaved changes. Save your work first if needed.
2
Set Up the Board Position
Clear the board and place exactly the pieces you want for your puzzle

You can start from the standard position, or clear the board and build a custom position from scratch. Use the piece palette in the left sidebar to select pieces, then click any square to place them. Right-click a square to erase a piece from it.

  • 1Click ✕ Clear Board to remove all pieces (for a custom position)
  • 2— or — ⟳ Reset to Start Position to restore the standard chess setup
After clicking "New Puzzle"
After clicking "Clear Board"
3
Place Pieces Using the Palette
Select a piece from the sidebar, then click the target square

The piece palette in the left sidebar shows all 6 piece types in both colors. Click a piece to select it (it will highlight with a gold border), then click any board square to place it. To remove a piece, click the ✕ Eraser or right-click any square.

  • 1Click a piece in the White Pieces or Black Pieces palette row
  • 2The selected piece gets a gold highlight border
  • 3Click any board square to place the piece there
  • 4Repeat for each piece you want in the puzzle
  • 5To erase: click ✕ Eraser then click the square — or right-click any square directly
⚠️ Clicking the same square again with a different piece replaces the piece already there. Only one piece per square.
Example: White Queen on d1, White Pawn on e2 · Black King on e8, Black Pawn on d7
💡 Keep Anti-Chess in mind: you don't need both kings. Any combination of pieces is valid for a puzzle. Design positions where forced captures create an interesting sequence.
4
Set the Turn & Fill Puzzle Details
Choose who moves first, and give your puzzle a title and difficulty

Before recording, set which side moves first using the Side to move first toggle beneath the palette. Then fill in the puzzle details in the right panel: title, description, and difficulty rating.

  • 1Click ⬜ White or ⬛ Black in the Side to move first selector
  • 2In the right panel, type a Title for the puzzle
  • 3Optionally add a Description / hint text
  • 4Select a Difficulty from the dropdown (Beginner → Expert)
Puzzle details filled in — right panel
5
Start Recording the Solution
Locks the starting position and switches to Record Mode — rules are now enforced

Click ⏺ Start Recording to lock the current board position as the puzzle's start. The app enters Record Mode (red badge). From this point, all Anti-Chess rules are enforced — you can only play legal moves, and captures are mandatory.

  • 1Click ⏺ Start Recording in the Workflow section
  • 2The badge changes to ⏺ Recording (red)
  • 3The FEN of the start position is locked (shown below the board)
  • 4If any captures are immediately available, a red warning banner appears
Record Mode active — ⏺ Recording badge shown
⚠️ You need at least 2 pieces on the board before starting to record. The app will show an error toast if the board is too sparse.
6
Play the Solution Moves
Click a piece to select it, then click the target square — move hints shown

In Record Mode, click any of your pieces to select it. The board will show legal move indicators: dark dots on empty squares you can move to, and red rings on enemy pieces you can capture. If a capture is available anywhere, you must capture (the red banner reminds you).

  • 1Click a piece of the side whose turn it is — it highlights green
  • 2Legal move targets appear: dots (moves) and red rings (captures)
  • 3Click a target square to make the move
  • 4The move is logged in the Solution Moves list (left sidebar)
  • 5Last-move squares highlight in yellow (Lichess style)
  • 6Repeat for each move in the solution — alternating sides automatically
Piece selected — legal moves shown
After first move — yellow last-move highlight
Black's turn — select a piece
Solution list growing as moves are recorded
ℹ️ Pawn Promotion: When a pawn reaches the last rank, a promotion dialog pops up. Click any piece (including King — allowed in Anti-Chess!) to promote.

Undo: Made a wrong move? Click ↩ Undo Last Move to step back.
⚠️ Mandatory captures: If a red warning banner appears at the top of the Solution Moves panel, at least one capture is available and you must capture before making any other move.
7
Stop Recording & Save the Puzzle
Finalise the solution and save it to local storage

When you have recorded all the solution moves, click ⏹ Stop Recording. The app returns to Setup Mode. Review the solution list, then click 💾 Save Puzzle in the right panel to persist the puzzle.

  • 1Click ⏹ Stop Recording when the solution is complete
  • 2The mode badge returns to ⚙ Setup Mode
  • 3Verify the Solution Moves list looks correct
  • 4Confirm the Title, Description and Difficulty in the right panel
  • 5Click 💾 Save Puzzle — a green toast confirms the save
  • 6The puzzle appears in the Saved Puzzles list below
Recording stopped — back to Setup Mode
Puzzle saved — toast notification + list updated
💡 Puzzles are saved to localStorage in your browser — they persist across page refreshes. For long-term storage, use 📤 Export All (JSON) to download a backup file.
Phase 2 — Viewing & Managing Puzzles
8
Load a Puzzle & Navigate the Solution
Click a puzzle in the list to enter View Mode, then step through moves

Click any puzzle in the Saved Puzzles list (right panel) to load it into View Mode. The board resets to the puzzle's start position. Use the navigation buttons in the left sidebar to step through the solution forward and backward.

  • 1Click a puzzle name in the Saved Puzzles list (right panel)
  • 2The board loads the puzzle's start position
  • 3The mode badge changes to ▶ View Mode
  • 4The Navigate Solution panel appears in the left sidebar
  • 5Click ▶ to advance one move, ◀ to go back
  • 6Click ⏮ to jump to start, ⏭ to jump to end
  • 7Click a move in the Solution Moves list to jump directly to that position
Puzzle loaded — View Mode, start position
After clicking ▶ — move 1 shown with yellow highlight
Move 2 highlighted in solution list
After ⏮ — back to puzzle start position
9
Export & Import Puzzles
Back up your puzzles as JSON, or export a single puzzle as PGN

The Import / Export section of the right panel lets you save puzzles to files and load them back. JSON is used for full collections; PGN is the chess standard format compatible with tools like Lichess.

  • 1📤 Export All (JSON) — downloads all saved puzzles as antichess_puzzles.json
  • 2📂 Load Puzzles (JSON) — opens a file picker to import a previously exported JSON file
  • 3📄 Export Current (PGN) — exports the currently active puzzle in standard PGN format with [Variant "Antichess"] header
  • 4🗑 Delete Puzzle — permanently removes the active puzzle after confirmation
ℹ️ The exported JSON is a plain array of puzzle objects. Each puzzle contains: id, title, description, difficulty, startFEN, and solution (an array of { uci, san, fenAfter } objects).

Tips & Best Practices
🎯
Good puzzle design Create positions where only one sequence of captures leads to losing all pieces quickly.
↩
Use Undo freely During recording, Undo removes the last move from the solution list. Experiment without fear.
💾
Export regularly localStorage can be cleared by the browser. Export JSON backups regularly.
📋
FEN box The FEN below the board always shows the current position. Copy it for external tools.
🖱️
Right-click to erase In Setup Mode, right-clicking any square instantly removes the piece on it.
⚙️
Edit saved puzzles Load a puzzle, switch back to Setup, re-record, and save — the existing entry updates.
